# 📝 Smart Online Exam System

A web-based online examination system developed using Node.js, HTML, CSS, and JavaScript.

## 🚀 Features

- 🔐 User Registration & Login
- ⏱️ Timer-based Exam
- 📊 Progress Bar
- 📌 Question Sidebar Navigation
- 🚫 Tab Switch Detection
- ⚡ Auto Submission
- 📈 Instant Result with Score
- ✅ Pass/Fail Status

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js

## ▶️ How to Run

1. Open Command Prompt
2. Go to project folder:cd Desktop
cd online_exm_system
3. Run server: node server.js
4. Open browser: http://localhost:3000/login.html
